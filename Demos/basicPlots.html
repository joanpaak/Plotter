Tests and examples

Take a gander at these examples to see how various common plottinh tasks are accomplished. See the Contents list below to get an idea what you find from each of the examples.

Contents

  1. Scatterplot
    • Demonstrates a basic x/y scatterplot
  2. Emtpy plot
    • How to create an empty plot in which to maybe draw something later.
  3. Empty plot with x label
    • How to add x axis label
  4. Empty plot with y label
    • How to add y axis label
  5. Empty plot with x and y labels
    • How to add x and y axis labels
  6. Traceplot, theta on x axis
    • Plot with a line. Concretely shows how to plot a a Markov Chain when indices are on y axis. Kind of a 90 degree rotated version of what's typically done.
  7. Traceplot, theta on y axis
    • See above; a common way of drawing a Markov Chain.
  8. Large scatterplot
    • Times how long it takes to draw 10000 points. Mainly just to satisfy curiosity.
  9. Handling tickmarks
    • Demonstrates the option pruneLongTickmarks which tries to stop tick labels from overlapping.
  10. Omitting axes
    • How to draw a plot without axes and add a point onto the emtpy canvas.
  11. Drawing a vector as points/lines
    • You don't necessary have to provide both x and y coordinates to draw points/lines. If you provide just a vector, indices are created autoatically.

Handling tickmarks

Overlapping tickmarks are pruned by default. This first example uses the option pruneLongTickmarks to disable the feature.

                let test1 = new Plot(
                document.getElementById("test9a"),
                null,
                {
                    xlim : [0, 5],
                    ylim : [-5, 5],
                    ytickplaces : [0, 1],
                    ytickmarks : ["Long text", "Another long text"],
                    xtickplaces : [1, 2, 3, 4],
                    xtickmarks : [
                        "Text",
                        "This is a particularly long tick mark",
                        "Text",
                        "Text"
                    ],
                    pruneLongTickmarks : false
                }
            )

In this next plot the feature is left on:

             let test2 = new Plot(
                document.getElementById("test9b"),
